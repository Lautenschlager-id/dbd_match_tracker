# DBD Match History Tracker

### ⚠️ Limitations & Prerequisites
Before installing, please ensure your system and accounts meet the following requirements:
* **Operating System:** Windows only (the script relies on Windows Task Scheduler).
* **Authentication:** Google Single Sign-On (SSO) is the only supported login method.
* **Security:** Your Google Account **must** have Two-Factor Authentication (2FA) enabled.

---

## ⚙️ Configuration

1. **Initialize the Environment File**<br>
   Create a new file named `env.json` by copying the provided `env.sample.json`.

2. **Configure Your Credentials**<br>
   Open `env.json` and update the following fields:
   * `credentials.username`: Your Google email address.
   * `credentials.password`: Your Google password.
   * `credentials.otp`: Your Google 2FA TOTP Secret Key (see the dropdown below if you need help finding this).

<details>
  <summary><b>Step-by-Step Guide: How to Extract Your Google 2FA TOTP Secret Key</b></summary>
  <br>
  <ol>
    <li><b>Navigate to Security Settings:</b> Open your Google Account settings and go to the <b>Security</b> tab.</li>
    <li><b>Access 2-Step Verification:</b> Scroll to "How you sign in to Google" and select <b>2-Step Verification</b> (you may be prompted to log in again).</li>
    <li><b>Add Authenticator:</b> Under "Add more second steps to verify it's you," select <b>Authenticator app</b>.</li>
    <li><b>Reveal the Secret Key:</b> Click <b>+ Set up authenticator</b>. When the QR code appears, do <i>not</i> scan it. Instead, click the button that says <b>Can't scan it?</b> or <b>Use setup key</b>.</li>
    <li><b>Copy the Key:</b> Google will display a long, alphanumeric string (e.g., <code>abcd efgh ijkl mnop</code>). This is your <b>TOTP Secret Key</b>.</li>
    <li><b>Configure Credentials:</b> Copy this string, remove any spaces, and paste it into your configuration file as the value for <code>credentials.otp</code>.</li>
  </ol>
  
  > **Security Note:** This Secret Key provides full access to generate your 2FA codes. Treat it with the same level of security as your primary password.
</details>

3. **Configure Match Tracking Limits**<br>
   Set the `i_might_play_more_than_50_matches_in_a_day` flag to `true` or `false`. 
   * *Context:* 50 matches in a single day roughly equates to having the game open for 6 to 8 hours. 

   > **⚠️ BAN RISK WARNING:** 
   > If you set this to `true`, the tracker will run in the background while the game is open. **There is a risk that the game's anti-cheat may flag this background process as a threat.** The developer assumes no liability for any actions taken against your account.
   > 
   > Setting this to `false` is entirely safe. The process will only trigger *after* the game closes, meaning it will not interact with the running game or trigger anti-cheat software.

## 🚀 Installation

1. Locate the `install.ps1` file in the project directory.
2. Right-click the file and select **Run with PowerShell**.
3. A User Account Control (UAC) prompt will appear requesting Administrator privileges. Click **Yes/Accept** to complete the installation.
4. The installer also will automatically generate your personalized `generated_tampermonkey_[timestamp].js` file in the project folder.

---

## 🖥️ Interface Setup (Tampermonkey)

To view your match history, you need to import the script generated by the installer:

1. Install the [Tampermonkey Extension](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en) for your browser.
2. Navigate to your browser's **Extensions** menu > **Tampermonkey** > **Manage Extension** (or "Learn More").
3. Ensure both **Allow user scripts** and **Allow access to file URLs** are enabled.
4. Click the Tampermonkey icon in your browser toolbar and select **Create a new script...**
5. Delete the default code, and paste the entire contents of the **`generated_tampermonkey_[timestamp].js`** file created by the installer. Save the script (`Ctrl+S`).
6. The script is now configured and ready to track your matches!

## ✨ Features

The Tampermonkey interface injects a robust, feature-rich dashboard directly into your browser to aggregate and analyze your Dead by Daylight match history.

### 🕒 Match History & Pagination
Review individual match breakdowns in a highly structured, paginated log. 
* **Role Filtering:** Instantly isolate your match entries using the **All / Survivor / Killer** quick-toggle icons.
* **Granular Match Details:** Review end-game trial details at a glance—including your chosen character, total bloodpoints/score, item/addon configuration, perk loadouts, teammate/killer outcome summaries, map identity, exact timestamp, and match duration.
